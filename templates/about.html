About AI Test Script Generator

GTX 1660 optimized AI model for generating SWTBot test scripts from natural language descriptions

Project Overview

This project provides an AI-powered solution for automatically generating SWTBot test scripts from natural language test case descriptions. The system is specifically optimized for NVIDIA GTX 1660 series GPUs with 6GB VRAM, making it accessible for developers with mid-range hardware.

Using fine-tuned sentence transformers and advanced NLP techniques, the system can understand complex test scenarios and generate professional, executable SWTBot test code that follows best practices and Eclipse UI testing patterns.

Technical Specifications

Model Details
  • Base Model: paraphrase-MiniLM-L3-v2
  • Fine-tuning: Custom SWTBot dataset
  • Embedding Size: 384 dimensions
  • Training Time: ~3 seconds
  • Correlation: 0.88 Pearson, 0.87 Spearman
Hardware Requirements
  • GPU: GTX 1660/1660 Super/Ti (6GB VRAM)
  • RAM: 8GB+ recommended
  • Storage: 2GB for models
  • CUDA: 12.1+ support
  • Python: 3.8+
